## 应用信息

### 应用名称

**TransX** - Professional Localization Tool for Xcode Projects

专为 Xcode 项目设计的专业本地化翻译工具。

### 版本信息

关于页面显示以下版本相关信息：

- **应用版本（Version）**: 当前安装的 TransX 版本号，遵循语义化版本规范（Semantic Versioning）
  - 格式：`主版本号.次版本号.修订号` （例如：1.5.2）
  - 主版本号：重大功能更新或不兼容的 API 变更
  - 次版本号：向后兼容的功能新增
  - 修订号：向后兼容的问题修复

- **构建版本（Build）**: 内部构建编号，用于追踪具体的构建批次

- **在线文档**: 访问完整的使用文档和最佳实践指南

### 问题反馈

如果您遇到问题或有功能建议：

1. **Bug 报告**: 通过内置反馈功能或 GitHub Issues 提交
   - 描述问题的详细步骤
   - 附上屏幕截图或错误日志
   - 说明系统环境和应用版本

2. **功能请求**: 提交新功能建议
   - 描述期望的功能和使用场景
   - 说明该功能如何提升工作效率

3. **技术支持**: 发送邮件至技术支持团队
   - 紧急问题可获得优先响应
   - 提供详细的环境信息和问题描述
