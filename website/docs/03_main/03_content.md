
# 内容区

![](./imgs/img16.png)

内容区是 TransX 主窗口中最核心的工作区域，专注于多语言字符串的批量管理、编辑与质量控制。该区域以表格/列表视图集中展示当前选中的 .strings 文件或分组下的所有本地化条目，支持多语言对比、批量操作、筛选、排序等多种高效管理手段。

## 功能亮点

- **多语言并列展示**：以表格形式直观对比每个 key 在不同语言下的内容，便于发现缺失、差异和不一致。
- **多视图模式**：支持单语言/多语言切换、分组/全部视图、索引列、常量列等多种展示方式，满足不同场景需求。
- **强大筛选与搜索**：可按 key、内容、分组、翻译状态等多维度筛选，支持正则表达式和模糊搜索。
- **批量操作**：支持批量编辑、批量补全、批量移动、批量删除、批量导出等高效操作。
- **右键菜单丰富**：对任意条目右键可快速访问移动、复制、分组、导出、暂存、查找、跳转等常用功能。
- **内容高亮与标记**：缺失、冲突、重复、未使用等状态自动高亮，便于质量把控。
- **快捷键支持**：常用操作均可通过快捷键快速完成，提升效率。
- **分组与模块化管理**：支持按功能模块、界面结构等多维度分组，便于大型项目的结构化维护。

## 推荐工作流程

1. **浏览定位**：通过导航区选择目标 .strings 文件或分组，内容区自动加载对应条目。
2. **筛选查找**：利用顶部筛选/搜索功能，快速定位需要处理的 key 或内容。
3. **批量选择**：支持 Command/Shift 多选、全选，便于批量操作。
4. **右键操作**：对选中条目右键，选择移动、分组、补全、删除、导出等批量功能。
5. **内容编辑**：双击单元格或配合编辑区进行多语言内容的快速修改。
6. **高亮核查**：关注高亮标记的缺失、冲突、重复项，及时修正。
7. **分组管理**：根据项目需求灵活调整分组结构，保持内容有序。
8. **导出/同步**：处理完毕后可批量导出、同步到代码或交付翻译。

## 典型应用场景

- 大型项目的多语言内容集中管理与质量审核
- 新增/重构功能模块时的批量分组与整理
- 发布前的缺失内容补全与一致性检查
- 团队协作下的分工与批量任务分配
- 快速定位和修复翻译遗漏、冲突、冗余
