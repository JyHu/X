快速上手

TransX 是一款专为 Xcode 项目设计的专业多语言本地化管理工具，旨在提升 iOS/macOS 应用开发中的国际化工作效率。

概述

本工具全面支持 .strings 文件的创建、编辑、管理和维护，帮助开发者更轻松地处理项目中的多语言内容。通过直观的界面和强大的功能，TransX 让多语言管理变得简单高效。

核心功能特性

📁 项目管理

  • 多项目支持：可同时管理多个 Xcode 项目的本地化内容
  • 历史记录：自动记录最近打开的项目，支持快速访问
  • 项目配置：为每个项目提供独立的配置选项

📄 .strings 文件管理

  • 智能解析：自动扫描并解析项目中的所有 .strings 文件
  • 分组管理：支持使用 // MARK: - 标记对本地化条目进行分组组织
  • 多文件切换：在同一项目的多个 .strings 文件间快速切换

🔐 常量模式支持

  • 类型安全：为每个本地化 key 生成对应的常量，避免字符串拼写错误
  • 编译时检查：利用编译器检查常量名称的正确性
  • 多语言支持：兼容 Swift 和 Objective-C 项目
  • 灵活配置：支持 Normal、LocalizedStringKey、StaticString 等多种常量样式

✏️ 内容编辑与校验

  • 可视化编辑：提供直观的界面编辑多语言内容
  • 实时校验：支持对 key 和常量设置正则表达式校验规则
  • 空白字符处理：自动检测并清理隐藏的空白字符
  • 内容补全：可使用指定语言的内容快速补齐其他语言的缺失项

🔍 高级搜索与筛选

  • 多维度筛选：内容缺失、新建内容、改动内容、未使用的 key、完全重复内容、关联内容等
  • 灵活搜索：支持正则表达式、精确匹配等多种搜索模式
  • 批量操作：支持对筛选结果进行批量编辑、导出等操作

快速上手流程

  1. 启动应用，进入欢迎页面
  2. 新建项目：选择 Xcode 项目路径，配置筛选规则；或从历史记录中打开已有项目
  3. 配置项目参数：设置基础语言、配置空白字符处理、选择常量模式（推荐）
  4. 开始管理：在主窗口中浏览、编辑多语言内容，使用筛选功能定位需要处理的条目
  5. 导出保存：将修改导出到项目的 .strings 文件，如启用常量模式则同时导出常量文件

适用场景

  • iOS/macOS 应用的国际化开发
  • 多语言内容的集中管理与维护
  • 本地化内容的质量检查与审核
  • 团队协作的多语言项目管理
  • 现有项目的本地化重构与优化

💡 注意事项

  • 建议在进行大规模修改前做好项目备份或使用 Git 版本管理
  • 使用常量模式时，请确保常量文件路径正确配置
  • 导出操作会直接修改本地 .strings 文件，请谨慎操作
  • 建议定期使用筛选功能检查内容缺失和重复项
