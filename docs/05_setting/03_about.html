应用信息

应用名称

TransX - Professional Localization Tool for Xcode Projects

专为 Xcode 项目设计的专业本地化翻译工具。

版本信息

关于页面显示以下版本相关信息：

  • 应用版本（Version）: 当前安装的 TransX 版本号，遵循语义化版本规范（Semantic Versioning）

    • 格式：主版本号.次版本号.修订号 （例如：1.5.2）
    • 主版本号：重大功能更新或不兼容的 API 变更
    • 次版本号：向后兼容的功能新增
    • 修订号：向后兼容的问题修复

  • 构建版本（Build）: 内部构建编号，用于追踪具体的构建批次
