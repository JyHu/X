TransX - 概览

概述

TransX 是一款专为 Xcode 项目设计的专业多语言本地化管理工具,旨在提升 iOS/macOS 应用开发中的国际化工作效率。本工具全面支持 .strings 文件的创建、编辑、管理和维护,帮助开发者更轻松地处理项目中的多语言内容。

核心功能特性

1. 项目管理

  • 多项目支持: 可同时管理多个 Xcode 项目的本地化内容
  • 历史记录: 自动记录最近打开的项目,支持快速访问
  • 项目配置: 为每个项目提供独立的配置选项,满足不同项目的个性化需求

2. .strings 文件管理

  • 智能解析: 自动扫描并解析项目中的所有 .strings 文件
  • 分组管理: 支持使用 // MARK: - 标记对本地化条目进行分组组织
  • 多文件切换: 在同一项目的多个 .strings 文件间快速切换

3. 常量模式支持

  • 类型安全: 为每个本地化 key 生成对应的常量,避免字符串拼写错误
  • 编译时检查: 利用编译器检查常量名称的正确性
  • 多语言支持: 兼容 Swift 和 Objective-C 项目
  • 灵活配置: 支持 Normal、LocalizedStringKey、StaticString 等多种常量样式

4. 内容编辑与校验

  • 可视化编辑: 提供直观的界面编辑多语言内容
  • 实时校验: 支持对 key 和常量设置正则表达式校验规则
  • 空白字符处理: 自动检测并清理隐藏的空白字符(如零宽空格、不间断空格等)
  • 内容补全: 可使用指定语言的内容快速补齐其他语言的缺失项

5. 高级搜索与筛选

  • 多维度筛选:
    • 内容缺失条目
    • 新建内容
    • 改动内容
    • 未使用的 key
    • 完全重复内容
    • 关联内容(蛛网式查找)
  • 灵活搜索: 支持正则表达式、精确匹配等多种搜索模式
  • 批量操作: 支持对筛选结果进行批量编辑、导出等操作

6. 导入导出功能

  • 多格式支持: 支持 CSV、JSON 等格式的导入导出
  • 选择性导出: 可导出全部内容或指定选中的条目
  • 关联内容导出: 支持导出筛选出的关联内容组

7. 协作与效率工具

  • 脚本扩展: 支持使用 JavaScript 编写自定义脚本进行翻译、转换等操作
  • 翻译集成: 可调用翻译脚本自动翻译内容
  • 粘贴板智能解析: 支持从格式化文本快速填充多语言内容
  • 同步创建: 在多个 .strings 文件中同步创建相同的本地化条目

8. Git 分支管理

  • 分支检测: 自动检测当前 Git 分支状态
  • 分支切换提醒: 当 Xcode 项目切换分支时提供警告提示
  • 分支同步: 支持切换到当前分支并重新加载对应内容

9. 可视化与高亮

  • 语法高亮: 支持自定义正则规则高亮显示特定文本格式(如占位符)
  • 进度可视化: 实时显示每种语言的本地化完成进度
  • 搜索高亮: 搜索时高亮匹配的关键词

10. 安全机制

  • 回收站功能: 删除的条目会进入回收站,支持恢复操作
  • 暂存区: 临时存储需要批量处理的条目
  • 操作日志: 详细记录所有操作历史,便于追踪和审计

快速上手流程

  1. 启动应用: 打开 TransX,进入欢迎页面
  2. 新建或打开项目:
    • 新建项目: 选择 Xcode 项目路径,配置筛选规则
    • 打开项目: 从历史记录中选择或浏览本地项目
  3. 配置项目参数:
    • 设置基础语言
    • 配置是否去除空白字符
    • 选择是否使用常量模式(推荐)
  4. 开始管理:
    • 在主窗口中浏览、编辑多语言内容
    • 使用筛选功能定位需要处理的条目
    • 新建或编辑本地化字符串
  5. 导出保存:
    • 将修改导出到项目的 .strings 文件
    • 如启用常量模式,同时导出常量文件

适用场景

  • iOS/macOS 应用的国际化开发
  • 多语言内容的集中管理与维护
  • 本地化内容的质量检查与审核
  • 团队协作的多语言项目管理
  • 现有项目的本地化重构与优化

系统要求

  • macOS 操作系统
  • 对于使用常量模式的项目,需要 Xcode 开发环境

注意事项

  • 建议在进行大规模修改前做好项目备份或使用 Git 版本管理
  • 使用常量模式时,请确保常量文件路径正确配置
  • 导出操作会直接修改本地 .strings 文件,请谨慎操作
  • 建议定期使用筛选功能检查内容缺失和重复项

通过 TransX,您可以更加高效、准确地管理 Xcode 项目的多语言本地化内容,显著提升国际化开发的工作效率和代码质量。
